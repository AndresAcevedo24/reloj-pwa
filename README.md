# Reloj - PWA 

Aplicación web / móvil que muestra la hora en tiempo real del sistema.

# Cómo ejecutar el proyecto

1. Clonar el repositorio:

git clone https://github.com/TU-USUARIO/reloj-pwa.git

2. Entrar a la carpeta:

cd reloj-pwa

3. Instalar serve:

npm install -g serve

4. Ejecutar el servidor:

serve .

5. Abrir en navegador:

http://localhost:3000

# Instalación como app

- En Chrome: botón "Instalar"
- En iPhone: usar Safari → "Agregar a pantalla de inicio"

# Tecnologías usadas

- HTML
- JavaScript
- Service Worker
- Manifest PWA
